### Edit Online (Recommended)

1. Clone this repository
2. Install a LaTeX distribution (MiKTeX, TeX Live, or MacTeX)
3. Edit `main.tex` with your favorite editor
4. Compile to PDF:
   ```bash
   pdflatex main.tex
   ```
